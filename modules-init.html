初始化 Module

# 创建新项目目录
mkdir myproject
cd myproject

# 初始化 Go Module
go mod init github.com/username/myproject

执行上述命令后，会在项目根目录生成 go.mod 文件：

module github.com/username/myproject

go 1.21

go.mod 文件说明

  • module：定义模块路径，通常是仓库地址
  • go：指定 Go 版本
  • require：声明依赖及其版本
  • exclude：排除特定版本的依赖
  • replace：替换依赖为本地路径或其他版本
