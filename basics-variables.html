变量声明

Go 语言有多种声明变量的方式：

package main

import "fmt"

func main() {
    // 方式1：使用 var 关键字
    var name string = "Go"
    var age int = 10

    // 方式2：类型推断
    var language = "Golang"

    // 方式3：短变量声明（最常用）
    version := "1.21"

    // 方式4：声明多个变量
    var (
        x int = 1
        y int = 2
    )

    fmt.Println(name, age, language, version, x, y)
}
